The Hello plugin provides a user greeting and tutorial, an interactive
online tutorial for learning more about supybot.

The command list is:

hello or hi - for all users
tutorialone, tutorialtwo, and tutorialthree - for all users
commandhelp - explains the Supybot command structure
createuserhelp - for any user that would like to register their nick
newownerhelp - for new owners
registryhelp - to learn more about the registry used by all Supybot plugins
               This is required reading for customizing your bot.
